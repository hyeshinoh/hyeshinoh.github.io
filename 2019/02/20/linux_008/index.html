1. 도메인(Domain)

2. hosts file

  • 어떤 도메인이 어떤 ip인지 정보를 담은 파일
  • DNS 접속 전에 hosts file에 적혀있는 도메인의 ip를 보고 접속함
  • 파일 경로: /etc/hosts

3. DNS

  • cat /etc/resolv.conf: DNS 서버 ip 확인
  • 도메인 구입: Freenom 등 무료 도메인을 구입할 수 있는 서비스 존재
  • 사용할 ip를 구입한 도메인으로 DNS 서버에 등록
    • ip addr: ip 주소 열람
    • curl ipinfo.io/ip: 자신의 ip 주소 확인
  • host [domain]: [domain]의 ip 주소를 알려줌

4. 서브도메인 (Sub Domain)

  • 도메인 앞에 prefix로 서브도메인을 붙여주면 서로 다른 ip를 가리키도록 할 수 있음

5. DNS 동작원리

  • dig +trace [domain]: [domain]의 DNS 서버를 통한 IP 주소 확인 경로 추적
