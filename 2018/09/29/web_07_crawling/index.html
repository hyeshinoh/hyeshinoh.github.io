Web crawling 기초

Sep 29 2018 2. Data Science 공부>6) 웹 Web 2 minutes read (About 364 words)

1. Python을 사용한 crawling의 방법

아래의 네 가지 방법이 사용되는데, 가능한 한 간단한 앞 번호의 방법을 사용하는 것이 효율적

  1. requests
  2. requests + bs4(BeautifulSoup)
  3. selenium
  4. scrapy

2. Requests 패키지를 이용한 crawling

python의 requests 패키지를 이용해서 코드상으로 request를 보내고 reponse를 받을 수 있음

2.1 API가 제공되는 경우

  • API로 request를 던지고 JSON format으로 데이터를 받아 pandas/csv로 저장

2.2 API가 제공되지 않는 경우

  • request를 던지고 response로 받은 HTML 코드(문자열 형태)를 CSS selector를 이용해 특정 element의 정보만 가져와서 pandas/csv로 데이터 저장
  • BeautifulSoup: str을 CSS selector로 크롤링할 수 있게 만들어주는 패키지
    • HTML 문자열을 parsing해서 변환해주면 CSS selector로 데이터를 가져올 수 있음

3. Selenium 패키지를 이용한 crawling

  • request로 crawling이 불가능한 동적웹페이지에 사용
    • 버튼을 클릭했을 때 나타난 페이지의 특정 element 가져오기 가능
  • CSS selector나 xpath를 사용
    (강사님은 CSS selector가 더 쉽고 자주 이용)

4. Scrapy를 이용한 crawling

  • scrapy: python library for web crawling, 사용이 좀 더 복잡
  • CSS selector, xpath를 사용함
  • requests와 selenium을 모아서 전략적으로 사용하는 라이브러리
