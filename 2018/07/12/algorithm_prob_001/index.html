BAEKJOON 1단계 1001번 문제

https://www.acmicpc.net/problem/1001

문제

두 정수 A와 B를 입력받은 다음, A-B를 출력하는 프로그램을 작성하시오.

입력

첫째 줄에 A와 B가 주어진다. (0 < A, B < 10)

출력

첫째 줄에 A-B를 출력한다.

예제 입력 1

3 2

예제 출력 1

1

풀이

(1) input을 split해서 리스트로 받아 A - B를 계산
