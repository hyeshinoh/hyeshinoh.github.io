1.1 comment

  • 프로그래밍에서 comment(주석)이란 코드가 실행되지 않는 라인을 의미
  • 코드에 대한 설명을 적거나 잠시 실행시키고 싶지 않은 코드에 주석처리

    • 대소문자 구분
    • 사용가능 문자: 소문자(a~z)/대문자(A~Z)/숫자/특수기호(_)
    • 맨 앞에 __ 사용 지양
    • 맨 앞에 숫자 사용불가
    • 예약어 사용불가: print, def, class, True, False…
