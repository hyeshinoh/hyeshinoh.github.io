Python Function - Decorator

  • 코드를 바꾸지 않고 함수의 기능을 추가하거나 수정하고 싶을때 사용
  • 여러개의 함수를 작성하는데 공통된 코드를 뽑아서 묶어 사용할 수 있음
  • *args와 **kwargs를 이용하여 내부함수와 내부 인자로 사용
