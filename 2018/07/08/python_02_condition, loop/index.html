4.3 list comprehension

  • 반복되어 출력되는 리스트 형태의 데이터를 만들어주는 문법
  • for문보다 속도가 빠름 (갈수록 굉장히 중요한 부분!)
  • ls = [<vlaue> for <value> in <list> (if <condition>)]
  • 간단한 형태만 가능하지만 가능한 한 많이 쓰면 좋은 문법

4.3.2 활용
