3.1 pillow 패키지

  • python에서 사용하는 이미지 처리 패키지

    • mac: $ pip3 install pillow
    • window: 5.1 버전은 jupyter에 문제 없음
      • $ conda install -c anaconda pillow
      • $ conda uninstall pillow
